# Was E2 physikalisch misst

Wir bestimmen aus der Einteilchen-Dynamik eines **lokalen Tight-Binding-Hamiltonians** $H=-J\,A$ (Adjazenz $A$) die **erste Überschreitungszeit** $t_\varepsilon(d)$ einer Amplitude $|\psi_j(t)|$ an Zielen $j$ im Graphabstand $d$. Eine lineare Beziehung $t_\varepsilon(d)\!\approx\! d/v_\!*+b$ definiert die **Frontgeschwindigkeit** $v_\!*.$ Das ist exakt die Quantum-Walk/CTQW-Kausalfront: auf der 1D-Kette ist $U_{ij}(t)\propto J_{i-j}(2Jt)$ (Bessel) und die Front läuft **ballistisch** mit $v_{\max}=2J$.
Für lokale Vielteilchensysteme liefern **Lieb–Robinson-Bounds** einen **rigorosen** Effektiv-“Lichtkegel” $\|[A(t),B]\|\!\lesssim\!\exp[-\mu(d-v|t|)]$ bereits **auf allgemeinen Metrik-Graphen** (keine Translationsinvarianz nötig) – genau unsere Bühne.

## 5) Kausalitäts-Bounds (Lieb–Robinson)

* **Existenz** eines endlichen Geschwindigkeits-Bounds $v_{\text{LR}}$ ist **rigoros** bewiesen – auch auf **allgemeinen Metrik-Graphen** (kein Gitter, keine Translation nötig).
* Grobe graphische Obergrenze (nur Heuristik): $v \lesssim O(J\,\deg_{\max})$. Hier $\deg_{\max}=6$ ⇒ $2J\,\deg_{\max}=12$ (J=1). Unser Messwert $v\sim 1.4$ liegt **weit darunter**.
* Für offene/Markovsche Dynamik existieren **verallgemeinerte LR-Bounds** (Quasi-Lokalität bleibt erhalten). ([arXiv][1], [Physical Review][2])

# Warum das jetzt „hart“ ist

1. **Goldstandard-Kalibrierung:** Die 1D-Kette zeigt (bekannt) $v_{\max}=2J$ via Bessel-Propagator—unser Pipeline-Design folgt exakt dieser Theorie.
2. **Graph-Allgemeinheit:** LR-Bounds auf **Metrik-Graphen** (Nachtergaele–Sims) sichern **kausale Fronten** unabhängig von Glattheit/Isotropie.
3. **Mehrfache Operationalisierungen:** $\varepsilon$-Scan, Richtungs-Resampling, Bootstrap-CIs, J-Skalierung → mehrere, **konsistente** Bestimmungen von $v_\!*$.
4. **Disziplinierte Unsicherheit:** Wir berichten **CIs**, $\varepsilon$-Abhängigkeit und Isotropie-Streuung explizit.
